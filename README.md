# Vue Dropfile

Drag-and-drop file upload component with Vue.js

![Drop zone output](./dnd.gif)

## Project setup

```bash
npm install
# OR
yarn install
```

### Compiles and hot-reloads for development

```bash
npm run serve
# OR
yarn serve
```

### Compiles and minifies for production

```bash
npm run build
# ORD
yarn build
```

### Lints and fixes files

```
yarn lint
```
